# 题目
给你一根长度为n的绳子，请把绳子剪成m段（m、n都是整数，n>1并且m>1），每段绳子的长度即为k[0],k[1],k[2]...k[m]。请问k[0] * k[1] * ... * k[m]的最大乘积是多少？
# 思路
1. (P14)
动态规划算法
- 每次取f(n)=max{f(i),f(n-i)}即可
2. （P14_2）
贪心算法P98，当n>=5时，我们有2(n-2)>n,并且3(n-3)>n,并且3(n-3)>=2(n-2),因此我们需要尽可能多的剪成长度为3的段，当剩下4时，剪成2*2的
